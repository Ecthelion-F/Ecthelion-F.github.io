在写堆优化的Dijkstra模板的时候用到了 STL 的priority_queue，队列中压入了自定义的结构体，需要重载运算符利用仿函数实现比较，鉴于自己太菜既没有查到也记不住原理，只好暴力记录用法待查。

重载运算符方法

这种方法在定义的结构体中重载运算符来实现大/小根堆，缺点似乎是没法混合生成大根堆和小根堆….？

//自定义的结构体，以len作为比较关键字
struct Node{
    int pos;
    int len;
    bool operator <( const Node &x )const{ //这里是重载的运算符
        return x.len < len;//产生小根堆
    }
};
priority_queue<Node> s;

食用方法

一直重载小于号运算符，接收到的参数有根的性质，小根堆就让参数在小侧，大根堆就让参数在大侧

在sort()函数中应用

比较通用的办法是实现一个比较函数cmp作为sort()的第三个参数传入

bool cmp(Node &a, Node &b){return a.len>b.len}
sort(s, s+n, cmp);

食用方法

  • 接收两个参数按顺序排列，降序用大于升序用小于
  • 另外重载运算符的食用姿势也可以，怎么理解我想不明白决定先老老实实用cmp
